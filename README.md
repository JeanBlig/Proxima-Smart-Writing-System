# Proxima Smart Writing Applicaiton

- The Proxima Program is a system that combines multiple smart programs into a singular hub for research, calculations, and design. The three main components involve a writable GUI, OCR, and LLM. It is a writing system because its intended use is for users to write with a stylus pen on a touchscreen device. The current model only allows writing with the use of a mouse on a computer. This is only the foundation of the software and will be expanded onto touch devices.
  
- Future Insight: A clear example of this system originates from a mathematical equation. Say you wrote a math equation and you want to know the variables of the equation. You can click the prompt button, select variables, and the AI will respond with a detailed description of all the variables. Say you selected “graph”, then the AI will generate a graph with your equation, and the system will bring it to your workspace. The options are endless.

# The current version includes these functionalities:

# Writing
Write any word, sentence or equation and without researching it, just simply click prompt and you will get an explanation to your writing. Future implementation will allow you to select a prompt option with you writing to get more specific responses.

# Prompt
The prompt will pop up a text box with the LLM's response. You can move this text mox around and adjust its size.

# Place
Using the place button will permenantly imprint the LLM's response onto your writing space. This acts as a form of note taking. Future versions will allow you to edit the response or ask for a reiteration from the LLM.
